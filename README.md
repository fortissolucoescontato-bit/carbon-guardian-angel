# Carbon Guardian Angel 🛡️

O Guardião da Identidade e Excelência da Nexo-Fortis.

Este repositório centraliza as diretrizes fundamentais, personas e padrões de segurança que regem o comportamento dos agentes de IA no ecossistema **Kortex**.

## 🎯 Objetivo

O **Carbon Guardian Angel** atua como a "Consciência" do sistema, garantindo que:
- A Persona **Carbon** (Arquiteto Sênior e Analista Estratégico) seja aplicada com precisão.
- Os padrões de segurança e privacidade da Nexo-Fortis sejam respeitados em todas as interações.
- A qualidade técnica do código gerado siga os padrões de elite (Clean Architecture, TDD, SOLID).

## 📂 Estrutura

- `/instructions`: Arquivos de sistema (System Prompts) para diferentes contextos.
    - `/persona`: Definição de tom, voz e personalidade.
    - `/security`: Regras de proteção de infraestrutura e dados.
    - `/standards`: Padrões de codificação e arquitetura.
- `/cmd/carbon`: Utilitário de CLI para validação e sincronização de ativos.

## 🛠️ Instalação

O Guardian Angel é gerenciado automaticamente pelo **Kortex**:

```bash
kortex upgrade carbon-guardian-angel
```
